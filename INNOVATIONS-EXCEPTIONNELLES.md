## 🎓 TECHNOLOGIES UTILISÉES

1. **CSS Transform 3D** : perspective, rotateX, rotateY, translateZ
2. **CSS Animations** : @keyframes avec cubic-bezier
3. **Backdrop Filter** : blur, saturate pour glassmorphism
4. **CSS Gradients** : linear-gradient, radial-gradient animés
5. **Box Shadow** : Multi-niveaux pour profondeur et néon
6. **Clip Path** : Pour révélation progressive
7. **Filter** : blur, hue-rotate pour effets holographiques
8. **React Hooks** : useState, useEffect, useRef pour interactivité
9. **Mouse Tracking** : Parallaxe basé sur position curseur
10. **Dynamic Rendering** : Particules générées en JS
