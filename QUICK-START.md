# ⚡ Démarrage Rapide - U.C.A.G

## 🚀 Déployer sur Vercel en 3 Minutes

### Étape 1: Ouvrir Vercel
👉 https://vercel.com

### Étape 2: Se Connecter
- Cliquez sur "Sign Up" ou "Log In"
- Choisissez "Continue with GitHub"

### Étape 3: Importer le Projet
1. Cliquez sur "Add New..." → "Project"
2. Cherchez `MakGroupDigital/ucag`
3. Cliquez sur "Import"

### Étape 4: Déployer
1. Vercel détecte automatiquement Next.js
2. Cliquez sur "Deploy"
3. Attendez 2-3 minutes ⏱️

### Étape 5: C'est Prêt ! 🎉
Votre site sera disponible sur:
```
https://ucag.vercel.app
```

---

## 📦 Développement Local

```bash
# Cloner le projet
git clone https://github.com/MakGroupDigital/ucag.git
cd ucag

# Installer les dépendances
npm install

# Lancer le serveur
npm run dev
```

Site local: http://localhost:9002

---

## 🔧 Scripts Disponibles

```bash
npm run dev          # Serveur de développement
npm run build        # Build de production
npm run start        # Serveur de production
npm run lint         # Linter
```
